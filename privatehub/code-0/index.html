Вы­ход с ко­дом 0 это ошиб­ка, свя­зан­ная с отсутствием файла ядра. Для ее ис­прав­ле­ния пе­рей­ди­те в уп­рав­ление сер­вер­ом

Зайдите во вкладку «Менеджер файлов»

Проверьте наличие ядра. Если оно есть, оно обязательно должно быть с имененем «server.jar»! Если у ядра другое имя, переименуйте его

Если ядра не было / это не помогло, переустановите ядро с официальных источников. Не забывайте: имя файла обязательно должно быть «server.jar»! Возможно, у вас появится ошибка 1. Как ее исправить мы писалитут
